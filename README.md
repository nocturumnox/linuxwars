# Death Star Escape Room - Linux Challenge

A Star Wars-themed Linux terminal escape room. Navigate through files, solve puzzles, and escape the Death Star using only Linux commands. Designed for educational fun and sysadmin skill practice.

---

## Features

- **File discovery challenges**  
- **Password cracking & privilege escalation puzzles**  
- **Log analysis with grep**  
- **Timed reactor puzzle**  
- **Escape pod pipeline challenge**  
- **Hundreds of decoy files to test your command-line skills**

---

## Requirements

- Linux system (tested on Ubuntu / Debian)  
- Bash shell  
- Root access to deploy the game  
- Recommended: run inside a VM or sandbox
